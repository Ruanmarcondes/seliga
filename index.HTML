se liga

É um programa do governo brasileiro que visa melhorar a qualidade das educações básicas, oferecendo apoio de professores para melhorar as notas que os alunos tiveram dificuldades durante o ano letivo.Em resumo o Se Liga escolar é uma das poderosa ferramenta que está transformando a maneira como os alunos sereparam para os estudos, com recursos personalizado, material didático de qualidade e uma da ótima comunidade virtual engajada.
